# run 뜻 정리: 왜 달리다, 작동하다, 운영하다가 다 run일까?

Q. 선생님, 왜 `run`은 사람도 run 하고 기계도 run 하고 가게도 run 해요?

영어 단어 중에 여러 뜻을 가진 단어들이 있죠?
그런데 그 뜻들을 잘 보면 하나의 이미지로 연결됩니다.
`run`은 그 연결이 아주 잘 보이는 단어입니다.

## 1) 한눈에 보는 핵심 연결

- `run` = 달리다
- `run` = 작동하다
- `run` = 운영하다

공통 느낌: `멈추지 않고 계속 움직이게 하다`입니다.

## 2) Step 1: 어원/형태 분해

`run`은 접두사와 어근으로 딱 잘라 설명하는 단어는 아닙니다.
오래된 영어에서부터 기본 이미지는 `움직이다`, `흐르듯 가다` 쪽이었습니다.
그래서 사람도 `run`하고, 기계도 `run`하고, 시스템이나 가게도 `run`할 수 있습니다.

## 3) Step 2: 대표 뜻별 설명

### ① 달리다

- 가장 기본 뜻은 몸이 빨리 움직이는 것입니다.
- 공통 이미지로 보면, 몸이 멈추지 않고 앞으로 계속 가는 느낌입니다.
- English: `She runs in the park every morning.`
- 한국어: 그녀는 매일 아침 공원에서 뛴다.

### ② 작동하다

- 기계, 프로그램, 엔진이 돌아가고 있는 상태에도 `run`을 씁니다.
- 공통 이미지로 보면, 내부 움직임이 계속 이어지는 것입니다.
- English: `The washing machine is still running.`
- 한국어: 세탁기가 아직도 돌아가고 있다.

### ③ 운영하다

- 가게, 회사, 프로젝트를 굴리고 관리하는 뜻으로도 씁니다.
- 공통 이미지로 보면, 어떤 시스템이 멈추지 않게 계속 움직이게 하는 것입니다.
- English: `She runs a small bakery with her sister.`
- 한국어: 그녀는 언니와 작은 빵집을 운영한다.

## 4) Step 3: 같은 어근/비슷한 패턴 단어

- `runner` / [ˈrʌnər]
뜻: 달리는 사람
느낌: `run + er`, 달리는 주체
예문: `The runner looked relaxed at the start line.`

- `running` / [ˈrʌnɪŋ]
뜻: 달리기, 작동 중인
느낌: 계속 움직이는 상태
예문: `We have running water again.`

- `rerun` / [ˈriːrʌn]
뜻: 재방송하다, 다시 돌리다
느낌: 한 번 간 것을 다시 run 시키기
예문: `They reran the show at night.`

- `outrun` / [ˌaʊtˈrʌn]
뜻: 더 빨리 달리다, 앞지르다
느낌: run에서 상대를 넘어가기
예문: `You can't outrun the storm.`

- `runway` / [ˈrʌnweɪ]
뜻: 활주로, 런웨이
느낌: run 하는 길
예문: `The plane waited on the runway.`

## 5) 마무리

이 단어는 결국 `계속 움직이게 하다`라는 이미지로 기억하면 쉽습니다.
사람이 직접 움직이면 `달리다`가 되고, 기계가 움직이면 `작동하다`가 되고, 조직이 움직이면 `운영하다`가 됩니다.
이 한 장면만 잡아두면 `run`의 뜻이 훨씬 덜 흩어집니다.
